# hytale-screen-script

Restart, Stop, Start.

Adapated [dotniphi/screen-server-scripts](https://github.com/dotniphi/screen-server-scripts) for Hytale.

hytale.service for systemd
  - replace with your info

server.sh in the folder above your /Server
